# programmieren
## Einleitung
Das Programm zeigt,welche Beeren man essen darf oder nicht. Der Benutzer kann die Beerenfarbe eingeben und dan sagt of sie giftig sind oder nicht. Je nach Beerenfarbe ist der Output anders.
